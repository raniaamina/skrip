# Add watermark to video

## Requirment
- `ffmpeg`
- `imagemagick`
- `exiftool`

## Position
Command | Description
--- | ---
C | Center
TR | Top Right
TL | Top Left
BR | Bottom Right
BL | Bottom Left

## How to use
```
$ ./skrip.sh [FILENAME] [WATERMARK TEXT] [POSITION]

$ ./skrip.sh Tutorial_Android_Programing.mp4 "Property of YourSoftwareHouseName" TR

```
